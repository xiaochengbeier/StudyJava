## 面相对象逻辑性，模拟万物待跟进。
    java 是一门面向对象的编程语言模拟万事万物之间的关系
## 封装继承复用性，多态操作灵活进。
    封装继承多态是java的三大特性 让其更加灵活 复用性更强
## 编译运行独立性，不与平台兼容进。
    跨平台是java的另一个特性和其编译的 .class 文件运行在jvm虚拟机上和什么平台操作系统没有关系 
    只要jvm跨平台即可

    学习编程语言 一定要静得下心 持之以恒
